Css3-carousel

Small and simple carousel with music in CSS3 and pure JS

Download this project as a .zip file Download this project as a tar.gz file

Welcome to My Simple Carousel.

This project is just a simple, easy to create or use carousel written in Pure JS and CSS3. No libraries, no plugins, nothing. However, in it's current stage it's also very simple, not overly sophisticated and not too pretty. That might change in the future. Here's how to use it

<link rel="stylesheet" type="text/css" href="css/carousel.css">
<script src="js/carousel.js" type="text/javascript"></script>
<script type="text/javascript">
	window.addEventListener('load', function()
	{
		var element = document.getElementById('carousel');
		carousel.init(element,
		[
			['images/1.jpg', 'music/1.mp3'],
			['images/2.jpg', 'music/2.mp3'],
			['images/3.jpg']
		]);
	});
</script>

And here is a live example. Use the left and right arrow keys to change photos, and the space or enter key to play the sound associated with it
